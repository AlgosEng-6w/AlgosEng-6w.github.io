    ZhiXi: BERT-Based Intelligent Proposal Analysis System NLP Algorithm Development

    2023.06 – 2024.01

    Overview: An automated semantic analysis system for large-scale public policy comments and legislative proposals, leveraging deep learning throughout the full pipeline — from web crawling and text preprocessing to model inference. The core module uses a pretrained BERT model to perform high-accuracy sentiment classification, multi-label topic tagging, and constructive suggestion extraction, providing policymakers with a real-time public opinion dashboard.

    Responsibilities:

    • Multi-task Model Design: Loaded bert-base-chinese via Hugging Face and designed a shared-backbone model supporting both three-class sentiment classification and multi-label topic recognition simultaneously.
    • Constructive Suggestion Extraction: Leveraged BERT's dense contextual embeddings combined with custom rule filters and intent classifiers to surface high-value proposals from large volumes of public comments.
    • Long-document Processing: Developed a Sliding Window segmentation and result aggregation algorithm to handle lengthy policy proposals, resolving semantic truncation caused by BERT's input length limit.

    Technical Highlights:

    • Robustness under Class Imbalance: Introduced Focal Loss during fine-tuning with dynamic sample-weight adjustment, improving recall on negative sentiment and contradictory opinion samples by 15%, effectively addressing the extreme data imbalance in civic data.
    • Deep Semantic Awareness: Compared to TextCNN and other baselines, BERT's bidirectional Transformer architecture better captures irony, negation, and other complex linguistic patterns, achieving over 90% sentiment analysis accuracy.
    • Long-text Semantic Integrity: Overlapping sliding windows ensure that key claims in lengthy proposals are not lost, improving constructive suggestion identification accuracy by approximately 8%.
